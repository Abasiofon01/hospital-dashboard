# Hospital Dashboard

A modern, responsive hospital management dashboard built with **React**, **TypeScript**, and **Tailwind CSS**. This application displays and manages hospital information for Nigerian healthcare facilities with advanced filtering, search, and pagination capabilities.

---

## Live Demo

- **Deployed Application:** [https://hospital-dashboard-kohl.vercel.app/](https://hospital-dashboard-kohl.vercel.app/)
- **GitHub Repository:** [https://github.com/Abasiofon01/hospital-dashboard](https://github.com/Abasiofon01/hospital-dashboard)

---

## Features

### Core Features

- ✅ Display 30 most recent hospitals in Nigeria (10 per page)
- ✅ Detailed hospital information view with modal
- ✅ Advanced search functionality (by name, address, state)
- ✅ Full pagination with customizable items per page
- ✅ Loading states with spinner animation
- ✅ Error handling with user-friendly messages
- ✅ Responsive design for all screen sizes

### Bonus Features

- Dark/Light mode toggle
- Debounced search for optimized performance
- Customizable page size (10/20/30/40 items)
- Clean, modern UI matching Figma design
- Type-safe implementation with TypeScript

---

## Tech Stack

### Frontend

- **React 18+** – UI framework with functional components
- **TypeScript** – Type safety and improved developer experience
- **Tailwind CSS** – Utility-first CSS framework
- **Lucide React** – Modern, customizable icon library

### State Management

- **Custom Zustand-like Store** – Lightweight, scalable state management

### API Integration

- **Fetch API** – Native browser API for HTTP requests
- **REST API** – Sofiamatics backend API integration

## Installation

### Prerequisites

- Node.js 20.x or higher
- npm or yarn package manager

### Setup Instructions

```bash
# Clone the repository
git clone https://github.com/Abasiofon01/hospital-dashboard.git
cd hospital-dashboard

# Install dependencies
npm install
# or
yarn install

# Start the development server
npm start
# or
yarn start

# Build for production
npm run build
# or
yarn build
```
